INTRODUCTION

Idea

PecuLab 隨機抽樣出三類資料

(其中2,3為我們重新設計的模型產出的假資料)

1. 真實的市場價格

2. local adversarial attack model output

3. Conditional AutoEncoder output

拋出給大家挑戰看看人類有沒有能力辨別出資料的真偽， 以驗證 PecuLab 所發展出的 financial vision data augmentation 是否有效

Data augmentation importance

因為在進行深度學習訓練時

需要Big-Data以確保訓練時不會產生over-fitting的現象

但資料是驅使AI引擎全力運轉的新石油

個人開發者或普通公司很難擁有或者搜集完整需要的資料

因此使用Data augmentation（資料增強）技術

以補足資料不足而導致的over-fitting問題

Game rules

遊戲共有20題，每次會出現2張圖片

其中一個是真實資料的K線圖

另一個則是假資料

各類別皆隨機出現，位置不固定

玩家只要選出真實資料的圖就得分

此遊戲考驗對K線圖的熟悉程度，亦為重要實驗資料

玩家可以仔細思考後做出判斷
